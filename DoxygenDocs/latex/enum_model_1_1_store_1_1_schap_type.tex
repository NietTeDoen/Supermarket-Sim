\doxysubsection{Detailed Description}
Enum die verschillende soorten schappen beschrijft. Elk type heeft een eigen afbeeldingspad, breedte en hoogte. Wordt gebruikt om verschillende schapafbeeldingen en groottes te beheren in de simulatie.
