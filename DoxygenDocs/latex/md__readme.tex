De Supermarkt Simulator is een Java 2D-\/simulatie waarin klanten en medewerkers automatisch rondlopen in een supermarkt. Klanten pakken producten uit schappen, plaatsen deze in een winkelmand/wagen en gaan naar de kassa. Medewerkers vullen schappen bij. Het programma draait volledig zelfstandig.\hypertarget{md__readme_autotoc_md3}{}\doxysubsection{\texorpdfstring{🎯 Doel}{🎯 Doel}}\label{md__readme_autotoc_md3}
Dit project is ontwikkeld als schoolopdracht met als leerdoelen\+:
\begin{DoxyItemize}
\item Werken met Java Swing (2D GUI).
\item Implementeren van pathfinding voor klanten en medewerkers.
\item Documenteren met UML-\/diagrammen.
